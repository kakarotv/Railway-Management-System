# Railway-Management-System
1.Project Description: - 
    We have created a railway database management system. 
    With the help of which the user can do following tasks: 
    a) User registration/login/logout 
    b) Ticket booking 
    c) Ticket cancelling 
    d) Check PNR status 
2.Develop Environment:- 
    We have used MySQL for the database designing and php in backend 
    to establish connection between database and frontend, and to run 
    various SQL queries. For frontend we have used HTML,CSS and 
    Javascript. 
3.Design process: 
    1) Conceptual design, including the structure of the whole system: 
    how many tables there are, what is the specific content of each 
    table, etc. Draw the whole E-R diagram. 
    2) Writing SQL codes to satisfy the conceptual design. 
    3)Creating frontend using html css and javascript
    4) Writing php code to (which setup connection between frontend 
    and backend) 
    5)test and debug
